Simpsons E-commerce Website Wireframe

Details

Project Type Personal Project
Platform Website Wireframes
My Role UI/UX Designer
Tools Canva

Description

This project is a high-fidelity wireframe for a TV show–themed e-commerce website inspired by The Simpsons. The design leverages imagery of main characters to create a visually engaging and cohesive experience, with thoughtful touches such as a character-themed 404 error page to maintain brand personality throughout the site.

The project focused on refining my UI design skills, including layout, typography, and visual hierarchy, while exploring how a strong creative direction and theme can enhance user engagement and overall usability.

Glam Muse

Details

Project Type Personal Project
Platform Website Wireframes
My Role UI/UX Designer
Tools Canva

Description

This high-fidelity wireframe was designed for a women’s fashion store, with the goal of making online shopping easy and enjoyable. I focused on clean navigation, clear product layouts, and intuitive filters so users can quickly find what they’re looking for.

The design reflects modern fashion trends while keeping the experience simple, smooth, and visually appealing. This project helped me grow my wireframing and UX skills, teaching me how to balance aesthetics with usability in a real-world scenario.

MAI Home Decor

Details

Project Type Personal Project
Platform Website Wireframes
My Role UI/UX Designer
Tools Canva

Description

These wireframes were created for a client who wanted a minimalistic e-commerce site for her personal artwork, using a strict colour palette of white, black, brown, and green.

The project challenged me to balance the client’s vision with my design expertise, creating a clean and functional interface that still felt unique and engaging. Through this project, I strengthened my skills in interpreting client needs, maintaining consistency, and delivering a visually cohesive design.
